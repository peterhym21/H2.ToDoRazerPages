# ToDo Projekt

### Opgave beskrivelse
-----

I dette projekt skal du lave en Razor Pages applikation der kan lave og manipulere en liste af ToDo´s

Det kunne være som følgende:

Opsætte menu <br>
Definere footer <br>
Finde billeder <br>
Oprette database <br>
Lave kontaktformular <br>
Osv. Osv. Osv.

Derefter skal man kunne vælge når et af punkterne er færdige.

Man skal endvidere også kunne rette og slette de forskellige ToDo´s



#### Forslag til de hurtige: <br>
Formatering med farve <br>
Sidde dato på så man kan sortere efter dato <br>
Vælge om man vil se færdige eller ej <br>

-----

### Mine krav
- [x] Struktur
- [x] Oprette et ToDo liste
- [x] Redigere de eksiterende ToDos
- [x] Sætte et ToDo til at være Færdigt
- [x] Slette ToDos
- [x] Navbar man kan vælge sider med
- [x] Partiel view til header og footer
- [x] News Letter side / partiel
- [x] Filtrer efter dato
